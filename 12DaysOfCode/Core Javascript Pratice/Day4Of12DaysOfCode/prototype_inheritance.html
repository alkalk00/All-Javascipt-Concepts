Inheritance Prototype In javascrpt

In Jvascript Inheritance can be only possible by prototype

When we create a fucntion name Mobile then in behind 2 objects are created:-

1. Mobile function with property of prototype 2. Prototype with protperty of __proto__

And with that function when we create object i.e. var m = new Mobile() then:-

1. m Object is created with __proto__ as protperty

This 2 objects goes to object.prototye and object.prototype goes to Null

Null is top of all

Again in same way when we create another function named Samgsung and it also create a object same like the upper one

These 2 objects are likned up with object.prototye and object.prototype is linked up with null

If we want to declar a prototype member in Mobile and want to access it into Samgsung it's not possible

Because these are two different constructor

If we want to access a instance member of Mobile constructor to Samgsung Constructor it is possible but prototype Inheritance is not possible

In this case only Prototype Inheritance can be possible

Note: we can go downward or back to Sangsung i.e. Mobile can't access the Sangsung constructor.The flow is always upward

======>

        function Mobile(){
            //instance memeber
            this.a = 10;
        }
        var m = new Mobile();

        //prototype member of Mobile
        Mobile.prototype.z = 30;

        //prototype memeber inheritance
        Samgsung.prototype = Object.create(Mobile.prototype);
        Samgsung.prototype.constructor = Samgsung;


        function Samgsung(){
            //instance memeber
            this.b = 20;
            Mobile.call(this);
        }
        var s = new Samgsung();

        //access instance member
        var mobile_prop = s.b;    //own property
        var x = document.getElementById("Mobile_prop");
        x=mobile_prop;
        document.write(" Sangsung property: "+x+"
");       //own property
        document.write("Mobile Instance member: "+s.a+"
");    //Mobile property
        
        //prototype inheritance
        var y = document.getElementById("samgsung_prop");
        var samgsung_prop = s.z;
        y=samgsung_prop;
        document.write("Mobile Prototype member: "+y);
